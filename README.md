# Agente Consultor de IA para Negocios

## Descripción
Agente de inteligencia artificial que analiza ideas 
de negocio usando consultoría estratégica profunda.

## ¿Qué hace?
- Analiza ideas de negocio en 4 fases
- Identifica problemas y oportunidades
- Da recomendaciones concretas
- Conversación continua con el usuario

## Tecnologías
- Python
- Groq API
- LLaMA 3.3 70B
